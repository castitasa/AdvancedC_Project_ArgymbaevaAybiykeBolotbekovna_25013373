# Movie Collection Organizer  
Advanced C Programming — Individual Project  

## Project Description
This project is a console-based Movie Collection Organizer written in C.  
It demonstrates dynamic memory allocation, pointers, modular programming, and file handling.  
Users can add, view, search, update, delete, save, and load movies.

/AdvancedC_Project_ArgymbaevaAybiykeBolotbekovna_25013373/
│
├── main.c
├── movies.c
├── utils.c
├── movies.h
├── movies_db.txt
└── README.md

## Features

- Add movie  
- List all movies  
- Search by title  
- Search by genre  
- Update movie  
- Delete movie  
- Save to file  
- Load from file  
- Dynamic array using realloc  
- No global variables  
- Modular code structure  

## Advanced Concepts Used

| Concept | Usage |
| Dynamic Memory | malloc, realloc, free |
| Pointers | MovieCollection*, dynamic strings |
| Structures | Movie, MovieCollection |
| Modular Programming | main.c, movies.c, utils.c, movies.h |
| File Handling | fopen, fgets, fprintf |
| No Globals | All data passed through pointers |

## File Format (movies_db.txt)

title<TAB>genre<TAB>year<TAB>rating
